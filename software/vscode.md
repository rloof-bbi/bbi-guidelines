# VSCode

Visual Studio Code (VSCode) is a free, open-source code editor developed by Microsoft. It provides powerful features such as syntax highlighting, debugging, intelligent code completion, version control integration, and a rich ecosystem of extensions. VSCode is lightweight, customizable, and supports development in many programming languages.

## Installation

1. Go to the [VSCode download page](https://code.visualstudio.com/download) and select the correct version for your operating system.
2. Download and run the installer.
3. You can keep the default setup options during installation.
4. After installation, launch VSCode from the Start menu or desktop shortcut.

## Basic Usage

-   Open a folder or file: `File > Open Folder...` or `File > Open File...`
-   Use the Command Palette (`Ctrl+Shift+P`) to access commands quickly.
-   Install extensions from the Extensions view (`Ctrl+Shift+X`).
